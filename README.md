## Запуск через .NET CLI

`dotnet run --project src/Calls.Web`

Интерфейс: http://localhost:5000


## Запуск в Docker

```ps
docker build -t calls .
docker run --rm -it -p 8080:8080 calls
```

Интерфейс: http://localhost:8080
